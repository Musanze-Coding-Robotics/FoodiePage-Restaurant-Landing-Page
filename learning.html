    Section 1: HTML Basics

    HTML is the structure of web pages. It uses tags like <h1>, <p>, and <a> to build content.

    <!DOCTYPE html>
<html>
<head>
  <title>My Page</title>
</head>
<body>
  <h1>Hello World</h1>
</body>
</html>

    Quiz: HTML Basics

    Which tag defines a paragraph?

    Section 2: HTML Page Structure

    Use tags like <header>, <main>, and <footer> to organize your page.

    Example structure:

    <body>
  <header>...</header>
  <main>...</main>
  <footer>...</footer>
</body>

    CSS controls the visual appearance of HTML. Use selectors and properties to change colors, spacing, and layout.

  font-family: Arial, sans-serif;
}

    Select elements using classes (.card) and IDs (#header) and apply style rules.

    .card {
  border-radius: 16px;
  box-shadow: 0 12px 25px rgba(31, 58, 136, 0.15);
}
