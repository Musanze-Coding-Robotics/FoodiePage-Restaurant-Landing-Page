    Introduction to HTML

    Not started

    Section 1: HTML Basics

    HTML (HyperText Markup Language) is the foundation of all web pages. It provides the structure and content that browsers display. HTML works by using tags to mark up content and tell browsers how to interpret it.

    Key Points:

    • <h1> to <h6> - Heading tags for titles and subtitles (h1 is largest, h6 is smallest)
    • <p> - Paragraph tag for text blocks
    • <a> - Anchor tag for creating links to other pages
    • <img> - Image tag to display pictures on your page
    • <div> - Division tag to group related content
    • <li> - List item tag for bullet points and numbered lists

    Example - Basic HTML Structure:

    <!DOCTYPE html>
<html>
<head>
  <title>My Page</title>
</head>
<body>
  <h1>Hello World</h1>
</body>
</html>

    Section 2: HTML Page Structure

    Semantic HTML tags help organize page content logically. These tags tell both browsers and developers what each section of the page represents. Using semantic tags makes your code more readable and improves accessibility.

    Important Semantic Tags:

    • <header> - Contains navigation, logos, and introductory content
    • <nav> - Contains navigation links
    • <main> - Main content area (use only one per page)
    • <article> - Self-contained content like blog posts
    • <section> - Thematic grouping of content
    • <aside> - Sidebar or supplementary content
    • <footer> - Contains copyright, links, and page information

    Complete Page Example:

    <body>
  <header>Logo and Navigation</header>
  <main>
    <article>Page Content</article>
  </main>
  <footer>Copyright Info</footer>
</body>

    CSS Fundamentals

    Not started

    Section 1: What is CSS?

    CSS (Cascading Style Sheets) is used to style HTML elements. While HTML creates the structure, CSS makes it look beautiful. CSS uses selectors to target HTML elements and properties to change their appearance.

    CSS Basics - Three Ways to Add Styles:

    • Inline CSS - Style directly in HTML tags
    • Internal CSS - Style in <style> tag in HTML head
    • External CSS - Link to separate .css file (best practice)

    Common CSS Properties:

    • color - Text color
    • background-color - Background color
    • font-size - Text size
    • padding - Space inside an element
    • margin - Space outside an element
    • border - Edge around element

    Example CSS:

    body {
  background-color: #f4f7ff;
  font-family: Arial, sans-serif;
  line-height: 1.6;
  color: #333;
}

h1 {
  color: #1f3a88;
  font-size: 32px;
}

p {
  font-size: 16px;
}

    Section 2: Selectors and Styling

    CSS selectors target specific HTML elements so you can apply styles. There are three main types of selectors: element selectors (tag names), class selectors (reusable), and ID selectors (unique).

    Types of CSS Selectors:

    • Element Selector - p { color: blue; } - Targets all paragraphs
    • Class Selector - .highlight { color: yellow; } - Use class attribute
    • ID Selector - #header { background-color: navy; } - Unique identifier
    • Group Selector - h1, h2, h3 { color: green; } - Multiple elements

    CSS Box Model (Important!):

    • Content - The actual element
    • Padding - Space inside the element
    • Border - Line around the element
    • Margin - Space outside the element

    Example with Advanced Styling:

    .card {
  border-radius: 16px;
  box-shadow: 0 12px 25px rgba(31, 58, 136, 0.15);
  padding: 20px;
  background-color: white;
  transition: transform 0.3s ease;
}

.card:hover {
  transform: translateY(-5px);
  box-shadow: 0 16px 30px rgba(31, 58, 136, 0.25);
}

    JavaScript Basics

    Not started

    Section 1: What is JavaScript?

    JavaScript is a programming language that runs in web browsers. While HTML creates structure and CSS adds styling, JavaScript adds interactivity and dynamic behavior to web pages. You can respond to user clicks, validate forms, update content, and much more.

    What Can JavaScript Do?

    • Respond to user interactions (clicks, typing, hovering)
    • Change HTML content dynamically
    • Modify CSS styles on the fly
    • Validate form data before sending to server
    • Create animations and effects
    • Store data in the browser (localStorage)
    • Make requests to servers (fetch API)

    Where to Put JavaScript:

    • In <script> tags in HTML (inline)
    • In separate .js files (best practice)
    • In HTML attributes like onclick

    First JavaScript Code:

    console.log('Hello World');

    Section 2: Variables and Functions

    Variables store data, and functions organize code into reusable blocks. Variables hold information (numbers, text, true/false), and functions let you run the same code multiple times without repeating yourself.

    Variable Declaration - Three Ways:

    • var - Old way, avoid using
    • let - Modern, block-scoped, can be changed
    • const - Modern, cannot be changed (preferred for constants)

    Data Types in JavaScript:

    • String - Text: 'Hello'
    • Number - Integer or decimal: 42 or 3.14
    • Boolean - True or false: true or false
    • Array - List of items: [1, 2, 3]
    • Object - Collection of properties: {name: 'John'}

    Function Examples:

    // Simple function
function greet(name) {
  return 'Hello ' + name;
}

// Arrow function (modern)
const add = (a, b) => a + b;

// Calling functions
console.log(greet('Alice'));
console.log(add(5, 3));

    Python for Beginners

    Not started

    Section 1: Python Basics

    Python is a beginner-friendly programming language known for its simple, readable syntax. It's used for web development, data science, artificial intelligence, automation, and more. Python emphasizes code readability, making it perfect for learning programming.

    Why Learn Python?

    • Easy to read and write - similar to English
    • Powerful libraries for different tasks
    • Used by companies like Google, Netflix, Spotify
    • Great for data analysis and automation
    • Perfect for beginners to learn programming concepts

    Python Basics:

    • Variables store data without needing types
    • Indentation (spaces) matters for code blocks
    • Comments start with # symbol
    • Lists hold multiple items: [1, 2, 3]
    • Dictionaries store key-value pairs: {'name': 'John'}

    Your First Python Code:

    name = 'Jessica'
print('Hello', name)

    Section 2: Python Functions and Control Flow

    Functions let you write code once and use it many times. Control flow statements like if/else and loops help you make decisions and repeat code. These are fundamental building blocks for all programs.

    Key Programming Concepts:

    • Functions - Named blocks of code you can call
    • Parameters - Information passed to functions
    • Return - Send result back from function
    • If/Else - Make decisions based on conditions
    • For Loop - Repeat code for each item in list
    • While Loop - Repeat code while condition is true

    Common Use Cases:

    • Automation - Run scripts to perform repetitive tasks
    • Data Processing - Analyze and transform data
    • Web Scraping - Extract data from websites
    • Building Bots - Create automated assistants

    Python Function Example:

    def greet(name):
  """Function that greets a person"""
  return 'Hello ' + name

# Call the function
print(greet('Alice'))

# Function with multiple parameters
def add(a, b):
  return a + b

print(add(5, 3))  # Output: 8

    SQL Essentials

    Not started

    Section 1: SQL Basics

    SQL (Structured Query Language) is used to communicate with databases. Databases store large amounts of organized data. SQL lets you ask questions about that data (queries), add new data, update existing data, and delete data. Nearly every website uses SQL.

    What Can SQL Do?

    • SELECT - Retrieve data from database
    • INSERT - Add new records to database
    • UPDATE - Modify existing records
    • DELETE - Remove records
    • WHERE - Filter results by conditions
    • ORDER BY - Sort results
    • JOIN - Combine data from multiple tables

    Database Structure:

    • Database - Container for all data
    • Table - Like a spreadsheet with rows and columns
    • Row - One record (e.g., one student)
    • Column - One field (e.g., student name)

    Your First SQL Query:

    SELECT * FROM students;

    Section 2: Filtering and Sorting Data

    As databases grow larger, you need to find specific data quickly. WHERE filters results by conditions, and ORDER BY sorts them. Together, these make powerful queries that answer real business questions.

    Filtering with WHERE:

    • = - Equal to
    • != - Not equal to
    • > - Greater than
    • < - Less than
    • >= - Greater than or equal
    • <= - Less than or equal
    • LIKE - Pattern matching
    • IN - Check if in list

    Advanced Queries:

    • COUNT - Count rows
    • SUM - Add up values
    • AVG - Calculate average
    • MAX/MIN - Find highest/lowest values
    • GROUP BY - Organize results into groups

    Complete Filtering Example:

    -- Find all high-performing students
SELECT name, grade FROM students
WHERE grade >= 80
ORDER BY grade DESC;

-- Count students by class
SELECT class, COUNT(*) as student_count
FROM students
GROUP BY class
ORDER BY student_count DESC;
