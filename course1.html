Introduction to HTML

Section 1: What is HTML?

HTML stands for HyperText Markup Language. It is the standard markup language for creating web pages. HTML describes the structure of a web page using markup.

Example:

<!DOCTYPE html>
<html>
<head>
    <title>My Page</title>
</head>
<body>
    <h1>Hello World</h1>
</body>
</html>

Section 2: Basic Tags

Common HTML tags include <h1> for headings, <p> for paragraphs, and <a> for links.

Example: <a href="https://example.com">Visit Example</a>
