CSS Fundamentals

Section 1: What is CSS?

CSS stands for Cascading Style Sheets. It is used to describe the presentation of a document written in HTML.

Example: body { background-color: lightblue; }

Section 2: Selectors and Properties

Use selectors like .class or #id to target elements. Properties like color, font-size control appearance.
