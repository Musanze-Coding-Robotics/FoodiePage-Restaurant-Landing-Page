JavaScript Basics

What is JavaScript?

JavaScript is a programming language that conforms to the ECMAScript specification. It is high-level, often just-in-time compiled, and multi-paradigm.

JavaScript is used for web development, server-side development (Node.js), and more.

JavaScript Output

JavaScript can display data in different ways: writing into an HTML element, writing into the HTML output, writing into an alert box, etc.

Example: document.write('Hello World');

JavaScript Statements

JavaScript statements are composed of: Values, Operators, Expressions, Keywords, and Comments.

Example: let x = 5;

JavaScript Variables

Variables are containers for storing data values. Use var, let, or const.

Example: let name = 'John';

JavaScript Operators

Operators are used to perform operations on variables and values. Types: Arithmetic, Assignment, Comparison, Logical.

Example: 5 + 3 = 8

JavaScript Data Types

JavaScript has dynamic types. Primitive types: string, number, boolean, undefined, null, symbol, bigint.

JavaScript Functions

A function is a block of code designed to perform a particular task. It is executed when "something" invokes it.

Example:

function myFunction() {
  console.log('Hello');
}

JavaScript Objects

Objects are variables too. But objects can contain many values. The values are written as name:value pairs.

Example: let person = {name: 'John', age: 30};

JavaScript Events

HTML events are "things" that happen to HTML elements. JavaScript can react to these events.

Example: onclick="myFunction()"

JavaScript Strings

Strings are used for storing and manipulating text. You can use single or double quotes.

Example: let text = "Hello World";

JavaScript Arrays

Arrays are used to store multiple values in a single variable.

Example: let cars = ['Saab', 'Volvo', 'BMW'];

Quiz: JavaScript Basics

What is JavaScript primarily used for?

  • A) Adding interactivity to web pages
  • B) Styling web pages
  • C) Structuring web content

Answer: A) Adding interactivity to web pages
